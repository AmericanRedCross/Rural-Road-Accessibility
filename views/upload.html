Upload zip file

 

Choose the correct zip file, press upload, wait for it to be processed and use the result in the next step.

The file you are uploading should be a zip file containing the following files all in the root of the zip file.

  • Shapefile of the road network; at least the .shp, .dbf & .shx files
  • Translation file to translate the shape-attributes to osm-tags; a .py file like this
  • Profile file to set speed and restrictions on the road network; a .lua file like this
After pressing upload zip file, the progress of uploading and processing is shown in the yellow status field. Once it is finished processing, step three will reveal itself.
